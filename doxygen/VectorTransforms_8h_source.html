28 // Entry point for unrolling declarative pattern rewrites.
29 // `op` is unrolled to the `targetShape` as follows, for each of its operands:
30 // 1. the unrolled type `unrolledVectorType` and number of unrolled instances
31 // `numUnrolledInstances` are computed from the `targetShape`. For now it is
32 // assumed the unrolling factors divide the vector sizes.
33 // 2. a fakeFork cast op is inserted that takes the operand and returns
34 // `numUnrolledInstances` results of type `unrolledVectorType`.
35 // 3. the original op is cloned `numUnrolledInstances` times, once for each
36 // result of the fakeFork cast op.
37 // 4. a fakeJoin cast op takes all these results and merges them into a single
38 // aggregate vector result whose size matches the original non-unrolled op
39 // operand types.
40 //
41 // Example:
42 //
43 // opA(operand0, operand1) // numUnrolledInstances = 3
44 //
45 // operand0 operand1
46 // | |
47 // fork fork
48 // <----------gather all fork ops --------->
49 // /|\ /|\
50 // f00 f01 f02 f10 f11 f12
51 // <---------- clone op 3 times --------->
52 // opA0(f00, f10), opA1(f01, f11), opA2(f02, f12)
53 // \ | /
54 // <-------------------- join ------------------------->
55 //
56 // Other local patterns then kick in iteratively (including DCE) and compose
57 // until all the fakeFork and fakeJoin ops are removed.
58 //
59 // This will be extended in the future to support more advanced use cases than
60 // simple pointwise ops.
61 SmallVector<Value, 1>
62 unrollSingleResultOpMatchingType(OpBuilder &builder, Operation *op,
63  ArrayRef<int64_t> targetShape);
