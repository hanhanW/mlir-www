Detailed Description

A BlockBuilder is a NestedBuilder for mlir::Block*.

This exists by opposition to LoopBuilder which is not related to an mlir::Block* but to a mlir::Value. It is meant to be used as a temporary object for representing any nested MLIR construct that is "related to" an mlir::Block*.
