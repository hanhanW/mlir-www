Detailed Description

Progressive lowering of a vector.contract a, b, c with row-major matmul semantics to a reduction_size-unrolled sequence: ``` at = vector.transpose a, [1, 0] bRow0 = vector.extract b[0] atRow0 = vector.extract at[0] c0 = vector.outerproduct atRow0, bRow0, c ...

bRowK = vector.extract b[K] atRowK = vector.extract at[K] cK = vector.outerproduct atRowK, bRowK, cK-1 ```

This only kicks in when VectorTransformsOptions is set to OuterProduct and the vector.contract op is a row-major matrix multiply.
