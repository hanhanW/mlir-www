Detailed Description

Represents a location that is external to MLIR.

Contains a pointer to some data structure and an optional location that can be used if the first one is not suitable. Since it contains an external structure, only optional location is used during serialization. The class also provides a number of methods for making type-safe casts between a pointer to an object and opaque location.
